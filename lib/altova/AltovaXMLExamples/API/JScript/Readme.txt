This directory contains JScript examples for the AltovaXML JScript API. This example is not intended to be pasted into HTML and cannot directly be used within the Altova Scripting Editor. Run it from a command prompt or directly from the Windows Explorer.

For details see the section "Application API" of the AltovaXML user documentation.

--------------------------------------------------------------------------
File list
---------------------------------------------------------------------------
Readme.txt			- this file

UseAltovaXML.js		- Start AltovaXML registered as automation server or connect to a running instance
					- Performs an XSLT transformation
---------------------------------------------------------------------------
Run the example from command line prompt
---------------------------------------------------------------------------
Open a command prompt window and type the name of one of the example scripts:

	UseAltovaXML.js

The Windows Scripting Host comming with all newer versions of Windows will execute the script and start-up AltovaXML.

---------------------------------------------------------------------------
Run the example from Windows Explorer
---------------------------------------------------------------------------
Double-click one of the example scripts. The Windows Scripting Host coming with all newer versions of Windows will execute the script within a command console. AltovaXML starts up. After the end of the script the command console gets closed automatically.
