This directory contains a VB example for the AltovaXML API. You can compile and run from within Visual Studio 2005 or Visual Studio 2008.

For details see the section "Application API" of the AltovaXML user documentation.

--------------------------------------------------------------------------
File list
---------------------------------------------------------------------------
Readme.txt				- this file

UseAltovaXML.csproj		- Visual Studio 2005 and 2008 project file
UseAltovaXML_VS2005.sln	- Visual Studio 2005 solution file
UseAltovaXML_VS2008.sln	- Visual Studio 2008 solution file

AltovaXMLexample.vb				- VB example source code

---------------------------------------------------------------------------
What the example does
--------------------------------------------------------------------------
The example starts up AltovaXML and checks if the given document is well-formed and performs an XSLT transformation.

You can modify the example in any way you like.

---------------------------------------------------------------------------
Compile and run the example with Visual Studio 2005
---------------------------------------------------------------------------
Double-click the file UseAltovaXML_VS2005.sln or open it from within a Visual Studio 2005 with 'File | Open | Project/Solution...'.

Use 'Debug | Start Debugging' or 'Debug | Start Without Debugging' to compile and run the example.

---------------------------------------------------------------------------
Compile and run the example with Visual Studio 2008
---------------------------------------------------------------------------
Double-click the file UseAltovaXML_VS2008.sln or open it from within a Visual Studio 2008 with 'File | Open | Project/Solution...'.

Use 'Debug | Start Debugging' or 'Debug | Start Without Debugging' to compile and run the example.
